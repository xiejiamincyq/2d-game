# 继续游戏快照格式

阶段 5 使用 `RunSnapshotStore.VERSION = 1`。正式运行默认写入 `user://five_minute_overdrive_run_v1.json`，headless 测试使用独立路径，不会覆盖玩家存档。

只允许在两种稳定边界写入：

- `wave_intro`：`pending_stage` 尚未开始，恢复后先显示该阶段提示。
- `settlement`：上一阶段已清剿、结算报价仍开放，恢复后继续领取或购买并关闭结算。

快照保存金币、三流派等级、卡牌层数、终极进化、稳定报价及事务号、玩家位置/生命/护盾、击杀数和用时。写入先落到临时文件，验证后再替换正式文件；非法数值、损坏 JSON 和未知版本都不会提供“继续游戏”入口。

敌人、子弹、传送门、拾取物、VFX、音频循环、超载临时状态和相机偏移从不进入快照。恢复始终创建一套新的运行时节点，再根据卡牌层数重建永久属性。
